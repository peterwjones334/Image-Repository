Help Page

Welcome to the Image Upload Application. Here is how you can use the application:

Uploading Images

  1. Navigate to the Upload page.
  2. Fill in the form with the required metadata:
    • Image: Choose the image file you want to upload. Allowed formats are png, jpg, jpeg, gif.
    • Part Number: Enter the part number associated with the image.
    • Version: Enter the version of the image.
    • Owner: Select the owner from the dropdown list.
    • Category: Select the category from the dropdown list.
    • Marking: Select the marking from the dropdown list.
  3. Click the "Upload" button to upload the image and save the metadata. Each uploaded image will be assigned a unique identifier (UID).

Browsing Images

  1. Navigate to the Browse page.
  2. Enter a query in the search bar to filter images based on part number, version, owner, category, marking, or leave it empty to see all images.
  3. Click on the filename link to view the image in a new window. In the new window, you can see the image and download it using the "Download" button.

Viewing Image Index

  1. Navigate to the Home page.
  2. See a list of all uploaded images along with their metadata including the UID.

Image Metadata

Each image uploaded to the application will have the following metadata fields:

  • UID: A unique identifier for the image.
  • Filename: The name of the uploaded file.
  • Date: The upload date of the image.
  • Part Number: The part number associated with the image.
  • Version: The version of the image.
  • Owner: The owner of the image, selected from a dropdown list.
  • Category: The category of the image, selected from a dropdown list.
  • Marking: The marking of the image, selected from a dropdown list.

If you have any questions or need further assistance, please contact support.
