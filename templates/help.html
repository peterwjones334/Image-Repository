Help Page

Welcome to the Image Upload and Viewer Application. This guide will help you navigate through the features of the application.

Uploading Images

To upload an image:

  1. Navigate to the Upload page.
  2. Fill in the form with the required metadata:
    • Image: Choose the image file you want to upload. Allowed formats are PNG, JPG, JPEG, GIF, TIFF.
    • Part Number: Enter the part number associated with the image.
    • Version: Enter the version of the image.
    • Owner: Select the owner from the dropdown list.
    • Category: Select the category from the dropdown list.
    • Marking: Select the marking from the dropdown list.
  3. Click the "Upload" button to save the image and its metadata. Each uploaded image will be assigned a unique identifier (UID).

Browsing Images

To browse and search for images:

  1. Navigate to the Browse page.
  2. Enter a query in the search bar to filter images based on part number, version, owner, category, or marking, or leave it empty to see all images.
  3. Click on a filename link to view the image in a new window.

Viewing and Downloading Images

To view and download images:

  1. When you click on an image filename from the browse page, it opens in a new window.
  2. In the new window, you can:
    • Use the "Fit to Window" button to fit the image to the window.
    • Use the "Center Image" button to center the image within the window.
    • Download the image in various formats:
      • Native Format: Download the image in its original format.
      • PNG: Download the image as a PNG file.
      • PDF: Download the image as a PDF file, fit to an A4 page.
      • JPEG: Download the image as a JPEG file.

Image Metadata

Each image uploaded to the application will have the following metadata fields:

  • UID: A unique identifier for the image.
  • Filename: The name of the uploaded file.
  • Date: The upload date of the image.
  • Part Number: The part number associated with the image.
  • Version: The version of the image.
  • Owner: The owner of the image, selected from a dropdown list.
  • Category: The category of the image, selected from a dropdown list.
  • Marking: The marking of the image, selected from a dropdown list.

Keyboard and Mouse Controls for Viewing Images

While viewing an image, you can use the following controls:

  • Zoom: Use the mouse wheel to zoom in and out.
  • Pan: Click and drag the mouse to pan around the image.
  • Keyboard Navigation:
    • Arrow Keys / WASD: Pan the image up, down, left, and right.
    • Page Up / Page Down: Zoom in and out.

Contact Support

If you have any questions or need further assistance, please contact support.
